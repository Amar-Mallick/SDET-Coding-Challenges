# SDET Coding Challenges & Test Automation

## 📖 About
A collection of coding challenges, test automation scripts, and API testing exercises utilizing tools like Java, Selenium, Playwright, and Postman. This repository serves as a practical portfolio demonstrating problem-solving skills and software quality assurance methodologies.

## 🛠️ Technologies & Tools
* **Programming Languages:** Java, JavaScript/TypeScript
* **UI Automation:** Selenium WebDriver, Playwright
* **API Testing:** Postman, Rest Assured
* **Build Tools & Management:** Maven, Jira

## 📂 Repository Structure
* `/Selenium-Java`: UI automation scripts and framework practice.
* `/Playwright-TS`: Modern web testing and automation scripts using JavaScript/TypeScript.
* `/API-Testing`: Postman collections and Rest Assured validation exercises.
* `/Java-Coding-Challenges`: Core Java logic building and problem-solving.
